# Közéleti Kiírás Automatizáló (ELTE GTK HÖK)

Ez az eszköz a közéleti ösztöndíjak adminisztrációjának egyszerűsítésére jött létre.

## Funkcionalitás
- **Adatfeldolgozás:** Hallgatói adatok importálása CSV fájlból.
- **Sablonkezelés:** Az adatok automatikus beillesztése a hivatalos sablonba.
- **Webes UI:** Felület a pontszámok kiosztására és a generálási folyamat vezérlésére.

## A projektről
A programot "szerelemprojektként" hoztam létre, hogy segítsem a Testület munkáját. Mivel nem vagyok hivatásos szoftverfejlesztő, a megoldások helyenként egyediek lehetnek, de a funkcionalitás tesztelt és működőképes. Bármilyen javítást vagy hozzájárulást szívesen fogadok!
